>> Running the server for handling socket events
	1. Change directory to node server directory
		
		cd node-server/

	2. Install required npm packages

		sudo npm install

	3. Run the node server.

		node main.js

>> Running local server for web application
	1. Go to the main folder.
	2. Run the local python server

			python -m SimpleHTTPServer <port_no>

			e.g python -m SimpleHTTPServer 9001

	3. In browser, go to address

		http://localhost:9001/messenger.html

Note : To test you will have to open several tabs in browser.
